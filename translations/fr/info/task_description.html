Vous avez reçu une lettre d'un ami que vous n'avez pas vu ni entendu depuis longtemps. Dans cette lettre, il vous donne des instructions pour trouver un trésor caché!

Dans cette mission, vous devez suivre une liste d'instructions qui vous mènera à un certain point sur la carte. Une liste d'instructions est une chaîne (string), chaque lettre de cette chaîne vous indique la direction de votre prochaine étape.

La lettre "f" - vous indique d'avancer, "b" - de reculer, "l" - de partir à gauche, et "r" - de partir à droite.

Cela signifie que si la liste de vos instructions est "fflff", vous devez avancer deux fois, faire un pas vers la gauche et avancer à nouveau deux fois.

Imaginons maintenant que vous vous trouviez à la position (0, 0). Si vous avancez, votre position deviendra (0, 1). Si vous avancez à nouveau, votre position sera (0, 2). Si votre prochain pas est vers la gauche, votre position deviendra (-1, 2). Après les deux pas suivants, vos coordonnées seront (-1, 4).

Votre objectif est de trouver vos coordonnées finales. Dans notre cas, il s'agit de (-1, 4).
