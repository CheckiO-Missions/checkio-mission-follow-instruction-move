在我们的例子中，计算每个方向的步数是指计算instructions 中每个唯一字母的个数。由于instructions 是一个字符串，您可以使用string.count() 方法，它可以实现这个功能！声明一个新变量，用method的值表示c 。 

instructions = "ffbrl"
c = instructions.count

好了，如何使用这个变量呢？

现在您可以调用此方法四次（每个方向使用不同的参数），并找出每个方向的总位移。 

instructions = "ffbrl"
c = instructions.count
print(c("r"), c("l"), c("f"), c("b"))

输出：

1, 1, 2, 1

如何将这些位移转换为最终坐标？

您应该考虑在同一轴上的相反方向。如果您前进四步，然后后退两步，您在这个轴上的最终移动是多少？

从描述中您可以看到，第一个坐标的正方向是右，负方向是左（当步幅为"l" 时，该坐标减小），第二个坐标的正方向是前，负方向是后（当步幅为"f" 时，该坐标增大）。

我应该进行怎样的精确计算？

您在每个坐标轴上的最终坐标是正负方向各自移动的差值。让我们声明两个变量并找出这些差值。 

axis_rl = c("r") - c("l")
axis_fb = c("f") - c("b")

那么，我的最终解决方案应该是怎样的呢？

您需要结合之前的提示，将整个代码封装为函数表达式，并返回存储最终坐标的两个变量的元组。

下面是整个解决方案：

def follow(instructions: str) -> tuple[int, int] | list[int]:

    c = instructions.count
    axis_fb = c('f') - c('b')
    axis_rl = c('r') - c('l')
    
    return axis_rl, axis_fb

在这个解决方案中，你在for 循环中进行计算，并更新预先声明的变量，同时考虑到每次移动的轴。 

def follow(instructions: str) -> tuple[int, int] | list[int]:

    move = {"f": 1, "b": -1, "l": -1, "r": 1}
    x = y = 0
    for i in instructions:
        x += move[i] * (i in "lr")
        y += move[i] * (i in "fb")

    return x, y

您可以使用lambda 表达式（匿名函数）和列表理解将所有逻辑压缩在一行中。 

follow = lambda i: [i.count(x)-i.count(y) for x, y in ('rl', 'fb')]
