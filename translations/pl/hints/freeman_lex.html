Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

💡 Wykonanie czterech kroków do przodu o długości 1 jest tym samym, co wykonanie jednego skoku do przodu o długości 4. Musi więc Pan policzyć całkowitą liczbę kroków w każdym kierunku, a następnie znaleźć dwie poprawne różnice między tymi liczbami.

Od czego powinienem zacząć?

W naszym przypadku policzenie liczby kroków w każdym kierunku oznacza policzenie liczby każdej unikalnej litery w instructions. Ponieważ instructions jest ciągiem znaków, można użyć metody string.count(), która robi dokładnie to! Proszę zadeklarować nową zmienną, niech będzie to c z wartością metody. 

instructions = "ffbrl"
c = instructions.count

Ok, jak używać tej zmiennej?

Teraz może Pan wywołać tę metodę cztery razy (z innym argumentem dla każdego kierunku) i sprawdzić całkowite przesunięcie w każdym kierunku. 

instructions = "ffbrl"
c = instructions.count
print(c("r"), c("l"), c("f"), c("b"))

Wynik:

1, 1, 2, 1

Jak przekształcić te przesunięcia w ostateczne współrzędne?

Proszę pomyśleć o przeciwnych kierunkach na tej samej osi. Jeśli wykona Pan/Pani cztery kroki do przodu, a następnie dwa kroki do tyłu, jakie jest ostateczne przesunięcie na tej osi?

Z opisu wynika, że pierwsza współrzędna ma kierunek dodatni w prawo i ujemny w lewo (gdy krok jest "l", ta współrzędna maleje), druga współrzędna ma kierunek dodatni do przodu i ujemny do tyłu (gdy krok jest "f", ta współrzędna rośnie).

Jakie dokładne obliczenia powinienem wykonać?

Pana końcowa współrzędna w każdej osi jest różnicą między odpowiednimi przesunięciami w kierunku dodatnim i ujemnym. Proszę zadeklarować dwie zmienne i znaleźć te różnice. 

axis_rl = c("r") - c("l")
axis_fb = c("f") - c("b")

Jak powinno wyglądać moje ostateczne rozwiązanie?

Proszę połączyć poprzednie wskazówki, zawinąć cały kod w wyrażenie funkcyjne i zwrócić krotkę dwóch zmiennych, które przechowują końcowe współrzędne.

Oto całe rozwiązanie:

def follow(instructions: str) -> tuple[int, int] | list[int]:

    c = instructions.count
    axis_fb = c('f') - c('b')
    axis_rl = c('r') - c('l')
    
    return axis_rl, axis_fb

W tym rozwiązaniu wykonuje Pan obliczenia w pętli for i aktualizuje wcześniej zadeklarowane zmienne, biorąc pod uwagę oś dla każdego ruchu. 

def follow(instructions: str) -> tuple[int, int] | list[int]:

    move = {"f": 1, "b": -1, "l": -1, "r": 1}
    x = y = 0
    for i in instructions:
        x += move[i] * (i in "lr")
        y += move[i] * (i in "fb")

    return x, y

Całą logikę można skompresować w jednym wierszu za pomocą wyrażenia lambda (funkcja anonimowa) i rozumienia listy. 

follow = lambda i: [i.count(x)-i.count(y) for x, y in ('rl', 'fb')]
